## Option Type Exercise

Below is a function that returns `null` when a user isn't found. Your task is to **rewrite this using the `Option` monad**, replacing `null` with `some` and `none`.

```ts
function getUserName(id: number): string | null {
  const users: Record<number, string> = {
    1: "Ahmad",
    2: "Ali"
  }

  return users[id] ?? null
}
```

Try rewriting it using `Option`, `some`, and `none`.

<AnswerDialog
  title="Answer"
  code={`import { Option, some, none } from "carbonteq-fp"

function getUserName(id: number): Option<string> {
  const users: Record<number, string> = {
    1: "Ahmad",
    2: "Ali"
  }

  return users[id] ? some(users[id]) : none()
}`}
/>

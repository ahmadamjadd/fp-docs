This guide will help you install and set up `@carbonteq/fp` in your TypeScript project.

---

## Install Carbonteq FP

You can install Carbonteq FP using your preferred package manager:

```bash
# npm
npm install @carbonteq/fp

# pnpm
pnpm add @carbonteq/fp

# yarn
yarn add @carbonteq/fp

# bun
bun add @carbonteq/fp
```

---

## TypeScript Configuration

To properly use Carbonteq FP, you'll need to configure your TypeScript project. Create or update your tsconfig.json with these settings:

```bash
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "nodenext",
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true,
    "strict": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true
  }
}
```
